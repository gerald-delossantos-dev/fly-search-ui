# FlySearch Angular Mockup

A polished, responsive flight-search industry UI mockup built with Angular standalone components.

## Requirements

- Node.js 18.19+ or Node.js 20.11+
- npm 10+

## Run locally

```bash
npm install
npm start
```

Then open the local Angular URL shown in the terminal, usually `http://localhost:4200/`.

## Production build

```bash
npm run build
```

The deployable files will be generated in `dist/flysearch-angular-mockup/browser`.

## Notes

This version pins all Angular packages to `18.2.13` to prevent npm peer-dependency conflicts caused by mixed Angular 20 minor versions.
